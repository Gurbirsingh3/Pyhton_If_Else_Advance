## 💡 Example Code

```python
temperature = float(input("Enter temperature: "))
pain = input("Enter pain level: ").lower()

if temperature > 102 or pain == "severe":
    print("High Priority")
elif 99 <= temperature <= 102 or pain == "moderate":
    print("Medium Priority")
else:
    print("Low Priority")
```
